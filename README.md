# Dotfiles v1.1

## Install Guide
Installs and updates all required programs and dependencies such as:
- Node
- Git
- Neovim
- Lazygit
- wget


And useful binaries:
- FFMPEG
- Svgo
- Rename

### Mac OS

```bash
$ sh install-programs-macos.sh
```
 
```bash
$ sh setup-symlinks.sh
```

### Linux
```bash
$ sh install-programs.sh
```
 
```bash
$ sh setup-symlinks.sh
```
