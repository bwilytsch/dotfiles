# Dotfiles v1

## Guide

### Setup

Managed via `Stow`

#### Mac

`brew instlal stow`

#### Ubuntu

`sudo apt instlal stow`

### Usage

Symlink file via
`stow <package_name>`

### Requirements

- tmux/tpm
- VimPlug

#### NPM Packages

Eslint
`npm install -g vscode-langservers-extracted`

Typescript
`npm install -g typescript-language-server`

Packages:

- Treesitter
- NERDTree
